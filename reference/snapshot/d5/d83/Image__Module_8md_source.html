59 @section making-a-image Making the image
60 
61 To create an Image object, we simply do the following:
62 
63 ```cpp
64 int width = 255;
65 int height = 255;
66 int bpp = 24;
67 Image myImage(FIF_PNG, "nameOfImage.png", ImageFlags::Write, width, height, bpp);
68 ```
69 
70 This code allocated space in memory for writing to a 255x255 image with a 24 bit depth.
71 
72 You can check the Image's documentation to understand better the needed arguments, but, for the sake of this tutorial, we'll explain it quickly:
73 
74 - `FIF_PNG` declares the [FreeImageFormat](http://graphics.stanford.edu/courses/cs148-10-summer/docs/FreeImage3131.pdf) (*page nº 13*) of the image (aka. it's extension)
75 - `"nameOfImage.png"` is the filename, including the extension
76 - `ImageFlags::Write` declares the image is only used to write.
77 - The rest of the arguments declare, respectively, `width, height` and `bpp` (these should only be given if the image is non existent/has only the write flag).
78 
79 Now that we have the image allocated, let's write to it.
80 
81 @section create-hue-image Creating the hue
82 
83 To create the hue, we use the following formula:
84 
85 ```cpp
86 for(int x = 0; x < width; x++) {
87  for(int y = 0; y < height; y++) {
88  red = Math::clamp(y-x, 0, height);
89  green = abs(y-height);
90  blue = x;
91  }
92 }
93 ```
94 
95 But how do we write these RGB values onto the image? For that, we'll use the Color class defined in the gengine-core module:
96 
97 ```cpp
98 GEngine::Color color;
99 ```
100 
101 After that, we simply apply the above formula:
102 
103 ```cpp
104 for(int x = 0; x < width; x++) {
105  for(int y = 0; y < height; y++) {
106  color.setRed((uint8_t)Math::clamp(y-x, 0, height));
107  color.setGreen((uint8_t)abs(y-height));
108  color.setBlue((uint8_t)x);
109 
110  myImage.drawPixel(x, y, color);
111  }
112 ```
113 
114 The three first instructions (inside the nested for loop) simply store the RGB values to the Color object. It's in the last instruction that we ask for Aurora to write at said pixel *(x, y)* said color.
115 
116 However, since this is a CLI application, we can't print the image to the console at runtime without *really* extra work. One way to see the results way easier is to simply save the image to disk.
117

130 At the start, the image is *lighter*. But why is this? Simple: the image on the top has *alpha* values, and we defined our image's bpp to be 24 (therefore, without alpha values).
131 
132 In fact, if you had noticed this in the middle of the tutorial and tried to add alpha, you may have caught **a lot** of warnings like these:
133 
134 ```
135 [WARNING] There are alpha values in the supplied color, however the image is not 32-bit.The alpha information will be lost. If you want to use alpha, first convert the image to 32-bit.```
136 
137 So, let's add alpha to our image.
138 
139 @section adding-alpha-image Adding alpha
140 
141 There are actually two ways for your image to allow alpha values. You either:
142 
143 - Declare the bpp to be 32 right in the constructor:
144 
145 ```cpp
146 int width = 255;
147 int height = 255;
148 int bpp = 32;
149 Image myImage(FIF_PNG, "nameOfImage.png", ImageFlags::Write, width, height, bpp);
150 ```
151 
152 Or you can also convert your image to 32 bits:
153 
154 ```cpp
155 myImage.convertTo32Bits();
156 ```
157 
158 After that, your image is ready to accept alpha values. To actually add them, just use the setAlpha() method of the Color class:
159 
160 ```cpp
161 for(int x = 0; x < width; x++) {
162  for(int y = 0; y < height; y++) {
163  color.setRed((uint8_t)Math::clamp(y-x, 0, height));
164  color.setGreen((uint8_t)abs(y-height));
165  color.setBlue((uint8_t)x);
166 
167  color.setAlpha((uint8_t)128);
168 
169  myImage.drawPixel(x, y, color);
170  }
171 ```
172 
173 The alpha values go from 0 to 255, being 0 totally transparent and 255 totally opaque.
174 
175 If now you change your code to add the alpha value, your image should now look exactly like the one on the start of the tutorial. Congratulations, you made your first image using Aurora!
